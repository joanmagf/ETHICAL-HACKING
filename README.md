# ETHICAL-HACKING


## How to install MKDOCS
### Requirements
```text
Python 3.12
PIP Package Manager
GitHub Account
```

### MKDOCS Installation on MacOS with Python
```bash
python3 -m venv venv
source venv/bin/activate
pip install mkdocs
pip install material
```

### Creating and launching MKDOCS site
```bash
mkdocs new .
mkdocs serve

mkdoscs build
mkdocs gh-deploy --clean
```
